This website project is a portfolio website. It is made out about, projects and contact me pages. The about page consist of 
a hero section including a introduction and a image with a button that redirects you to the project page. also it contains 
a skills section with cards one with soft skills and the other with technical skills. Follow by a Previous
employeers includes start date and end date, as well as transferable skills learned on the job. The project 
page has 4 cards 2 with projects name and some details, as well it includes 2 cards for future projects. The project cards 
include a view project button that redirects you to the corresponding github repository. The last page 
contains a table with contact information like email, phone, and linkedin and github, both having hyperlink references embedded.

This project used bootstrap as the main styling. Ways bootstrap were used in this project was to create the cards table and 
the background opacity. in the navigation bar bootstrap was used to make the navigation bar so when the website its display in
a small screen it trun into a hamburger collapsable menu.

The first function is a email validation email. It uses a regular expression to validate the format of an email address. It would 
ensure that the email contains characters before and after the "@" symbol and a valid domain format after the "." symbol. It could be 
integrated into any form submission process where email validation is required.

Function smoothScroll enables smooth scrolling to different sections of the page when navigation links are clicked. 
This functions can help improve the user experience on the website by adding form validation, dark mode, smooth scrolling, dynamic
project display, and modal popup
